Requirements
------------

::

    Django>=1.6
    django-appcore==0.0.4
    django-mptt==0.7.1
    django-guardian==1.2.5
    django-image-cropping==1.0.1
    django-markitup==2.2.2
    easy-thumbnails==2.2
    PyYAML==3.11
    Markdown==2.6.1

If you use Django<1.7, you should install South==1.0.2
